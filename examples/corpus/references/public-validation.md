# Public validation commands

Run `python3 eval/validate.py` to validate the public synthetic suite.
Run `python3 eval/validate.py --questions-root /path/to/private/questions --corpus-root /path/to/private/corpus` to validate a private canonical suite.

The validator accepts `--questions-root`, `--corpus-root`, and `--spec-root` flags.
